栄養素計算Webアプリ
3人チームで開発。

■ 担当部分（自分）

Pandasを用いた栄養データ集計処理

必要栄養素計算ロジックの設計・実装

BMI算出および必要エネルギー量算出処理

■ チームメンバー担当

レシピAPI連携部分

フロントエンド設計

■ 概要

身長・体重を入力すると必要栄養素を算出し、

選択した食材の合計栄養素との差分を表示するWebアプリです。

授業課題として開発しました。

■ 制作背景

日々の食事管理を簡単に行える仕組みを作りたいと考え制作しました。

PythonとWeb技術を組み合わせたアプリケーション開発を目的としています。

■ 使用技術

Python

Flask

Pandas

HTML / CSS

CSVデータ処理

■ 主な機能

身長・体重から必要栄養素を計算

食材データ（CSV）から栄養素を取得

複数食材の合計栄養素を自動計算

必要量との差分表示

楽天レシピAPI（現在APIキー期限切れのため停止中）

■ 工夫した点


日本語の表記揺れを正規化処理で統一

Pandasを用いたデータ集計の自動化

入力エラー時の例外処理を実装
